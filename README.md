## Config / param

  * `font-size` : `full|large|medium|small` (default: `large`)
  > * `full`: 100% of line height (widget height)
  > * `large`: 90% of line height (widget height)
  > * `medium`: 60% of line height (widget height)
  > * `small`: 40% of line height (widget height)
